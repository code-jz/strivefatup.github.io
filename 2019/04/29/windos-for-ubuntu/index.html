Windows子系统for Ubuntu

工具 Ubuntu Linux Windows 2019/04/29

Windows10 内置了Linux系统：WSL (Windows Subsystem for Linux, 又称Bash for Windows)。可以方便地在Win10里使用Ubuntu等Linux系统的命令行。

Linux对开发人员是友好的，我们的应用程序一般都是部署在Linux环境下的，今天将给大家介绍一款在windows下的开发利器，不再为想使用windows下面的工具，又想使用Linux环境开发而纠结了。

说明

系统要求：windows10，支持虚拟化。特别说明：windows10有多个版本，这里选用专业版。

准备

  1. 启用开发者模式
    打开设置 -> 点击更新和安全 -> 点击开发者选项 -> 启用开发人员模式 -> 重启电脑

  2. 更改系统功能
    点击应用和功能 -> 相关设置下的程序和功能 -> 选中应用或关闭Windows功能 -> 勾选适用于Linux的Windows子系统

install

  1. 打开应用商店(Microsoft Store)

  2. 搜索ubuntu

  3. 选择需要安装的Ubuntu版本，建议选用Ubuntu 18.04 LTS

进入Ubuntu系统

  1. 打开cmd，输入命令bash进入Ubuntu系统

  2. 建议安装cmder命令行工具，是windows下一款强大的bash工具

Hint

通过ls命令可查看目录结构

/mnt目录对应windows下面的目录

赶紧去试一下吧，让你的开发效率更快。
